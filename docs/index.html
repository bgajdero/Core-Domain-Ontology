Core Domain Ontology: Description back to ToC

Core Domain Ontology (CDO) is a semantic model developed to support the Informatics Domain Model IDM. The CDO is structured around five central nodes: objects, events, concepts, actions, and domain, which underpin various aspects of planning, reasoning, and execution in digital systems. It enables decentralized data representation, transformation, and alignment.
